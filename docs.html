Installation Guide

Quick Installation (Recommended)

⚠️ Important: Back up your important data before proceeding. AetherOS installation will overwrite the target drive.

Step 1: Download AetherOS

Download the latest AetherOS Community Edition ISO:

# Using wget
wget https://github.com/aether-os/releases/latest/download/aetheros-community.iso

# Using curl
curl -L -o aetheros-community.iso https://github.com/aether-os/releases/latest/download/aetheros-community.iso

# Verify checksum
sha256sum aetheros-community.iso

Step 2: Create Bootable USB

Linux/macOS:

# Find your USB device
lsblk

# Create bootable USB (replace /dev/sdX with your USB device)
sudo dd if=aetheros-community.iso of=/dev/sdX bs=4M status=progress
sync

Windows:

  • Download and install Rufus
  • Select the AetherOS ISO file
  • Choose your USB drive
  • Click "Start" to create bootable USB

Step 3: Boot from USB

  1. Insert the bootable USB into your target computer
  2. Restart the computer
  3. Press F12, F2, or Del (varies by manufacturer) to access boot menu
  4. Select the USB drive from the boot menu
✅ Success! AetherOS will boot in approximately 4.2 seconds and automatically detect your hardware.

Building from Source

Build AetherOS yourself for customization or contribution:

Prerequisites

# Install build dependencies (Ubuntu/Debian)
sudo apt update
sudo apt install build-essential nasm grub-common grub-pc-bin xorriso

# Install build dependencies (Fedora)
sudo dnf groupinstall "Development Tools"
sudo dnf install nasm grub2-tools xorriso

Build Process

# Clone repository
git clone https://github.com/aether-os/aether-os.git
cd aether-os

# Build kernel
make clean
make all

# Create bootable ISO
make iso

# Test in QEMU
make test

# Install to USB (replace /dev/sdX)
sudo make install USB_DEVICE=/dev/sdX

Build Configuration

# Debug build
make DEBUG=1

# Release build with optimizations
make RELEASE=1

# Build for specific architecture
make ARCH=x86_64
make ARCH=arm64

# Custom kernel features
make FEATURES="ai_agents gaming_mode dev_tools"

Troubleshooting

Common Issues

Boot Issues:
  • Ensure Secure Boot is disabled in BIOS/UEFI
  • Try different USB ports (USB 2.0 vs 3.0)
  • Verify ISO integrity with checksum
Performance Issues:
  • Check AI agent status: aether-agents status
  • Monitor system resources: aether-monitor
  • Review optimization logs: aether-logs performance
